# Image Analysis Examples

## Example 1: Camera Captured Scene

**Input**: Photo of outdoor landscape with mountains and forest

**Output**:
```
# Image Analysis: landscape.jpg

## Summary
A scenic outdoor landscape featuring mountains, forest, and clear sky

## Detailed Description
- **Primary subject**: Mountain range with forested slopes
- **Key objects**: Mountains, trees, possibly wildlife, sky with clouds
- **Colors**: Greens (forest), browns (terrain), blues (sky), whites (clouds)
- **Scene**: Outdoor, natural environment
- **Text content**: None visible
- **Lighting**: Natural daylight, likely morning or afternoon
- **Details**: Mountains show snow caps, forest is dense with layered vegetation

## Confidence Level
High
```

## Example 2: Document or Text Image

**Input**: Photo of a document with text

**Output**:
```
# Image Analysis: document.jpg

## Summary
Photograph of a document that appears to be a report or document with typed text

## Detailed Description
- **Primary subject**: Document page
- **Key objects**: Text content, lines, headings, possibly charts/tables
- **Colors**: White background, black text, document header/footer colors
- **Scene**: Indoor, document view
- **Text content**: [Describe any readable text]
- **Lighting**: Controlled lighting, evenly lit document
- **Details**: Document appears to have structured layout with sections

## Confidence Level
High
```
